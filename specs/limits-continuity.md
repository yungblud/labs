# 함수의 극한과 연속 — 고교 교과과정 정식 학습 (수학 II)

> 동기: 고등학교 교과과정(2007 개정, 이수 시기 2010~2012) 을 **단원별로 복습**한다.
> 위치: 「수학 II — ① 함수의 극한과 연속」. 미적분([`calculus-by-hand.md`](./calculus-by-hand.md),
> 다항함수 미분·적분)에서 **직관으로만 건너뛴 극한**을 교과서 수준으로 메꾼다.
> **개념 유도 + 유형별 문제풀이 둘 다** (교과과정 = 이해 절반 + 계산 훈련 절반).

---

## 목표

극한과 연속을, 미적분 체험 때의 "직관" 수준을 넘어 **교과서가 다루는 형식(좌·우극한, 부정형 계산법,
연속의 정의, 사잇값 정리)까지** 정식으로 한 바퀴 돈다. 개념을 유도하고, **각 유형을 손으로 계산**할 수
있게 되는 게 성공 기준.

```
① 극한의 뜻을 좌극한·우극한으로 정확히 정의한다        ← 존재 조건
② 극한의 성질(사칙연산)과 부정형 계산법을 익힌다        ← 0/0, ∞/∞, ∞−∞
③ 연속을 정의하고 불연속 유형·연속함수 성질을 정리한다   ← 극한값 = 함수값
④ 최대·최소 정리, 사잇값 정리와 그 응용(근의 존재)      ← 연속의 강력함
⑤ 각 파트 유형별 문제를 손으로 풀어 계산을 몸에 익힌다   ← 교과과정의 절반
⑥ "이해한 것 / 헷갈린 유형" 을 노트로 남긴다           ← 성공 판정
```

**성공 판정:** ⑥ 노트 + 유형별 문제를 스스로 풀 수 있으면 성공.

---

## 성격 — 이 실험에서 무엇이 "성공"인가

- **교과과정 따라가기가 목적.** 실무 연관성은 신경 쓰지 않는다(순수 학습).
- **개념만 ≠ 성공.** 교과서는 계산이 절반 → **유형별 문제풀이**를 반드시 포함.
- `ε-δ`(엄밀한 극한 정의)는 **고교 범위 밖** → 기본 제외, 원하면 선택 부록.
- 코드는 보조 — 극한처럼 눈으로 보면 확 오는 데(좌·우극한 수렴표, 사잇값 정리 근 찾기)서만 몇 줄.

## 단원 설계 — 무엇을 어떤 순서로 (교과서 목차 따름)

### 파트 1 — 함수의 극한
- 극한의 뜻: `x→a` 일 때 `f(x)→L` (a 에 "닿지 않고 다가감", a 에서의 함숫값과 무관).
- **좌극한·우극한**과 극한값의 **존재 조건**(좌극한 = 우극한).
- `x→∞`, `x→−∞` 의 극한(발산 포함).

### 파트 2 — 극한의 성질과 계산
- 극한의 사칙연산 성질(합·차·곱·몫).
- **부정형(不定形) 계산법:**
  - `0/0` 꼴 → 인수분해로 약분.
  - `∞/∞` 꼴 → 최고차항으로 나누기.
  - `∞−∞`, 무리식 → 유리화(켤레 곱).
- 함수의 극한의 대소 관계와 **압착(샌드위치) 정리** 소개.

### 파트 3 — 함수의 연속
- 연속의 정의: `lim_{x→a} f(x) = f(a)` (① 함숫값 존재 ② 극한 존재 ③ 둘이 같음).
- **불연속 유형**(구멍/점프/발산)과 그림.
- 연속함수의 성질(사칙·합성), 구간에서의 연속.

### 파트 4 — 연속함수의 정리
- **최대·최소 정리**(닫힌구간 연속 → 최대·최소 존재).
- **사잇값 정리(중간값 정리)** + 응용: 방정식 **근의 존재** 증명, 이분법.

### 문제풀이 세트 (`PROBLEMS.md`)
- 각 파트 유형별 예제 3~5개 + **단계별 풀이**. (좌우극한 판정 / 0-0꼴 / ∞-∞ 유리화 / 연속 판정·미정계수 / 사잇값 근 존재)
