## 무엇을 이해했나 (핵심 5줄)

1. **미분의 심장은 변화율, 엔진은 극한.** `f'(x)=lim_{h→0}[f(x+h)−f(x)]/h` — 두 점 기울기(심장)를
   한 점(순간)으로 좁히는 도구가 `lim`(엔진). 한 점에서 바로 재려 하면 `0/0` 벽 → 극한으로 우회.
2. **공식은 외우는 게 아니라 정의에서 떨어진다.** `(x+h)ⁿ` 의 앞 두 항만 살아남아
   `xⁿ → n·xⁿ⁻¹`. 상수배·합 규칙도 정의에 넣으면 자연히 분리됨.
3. **적분은 미분의 거울.** `lim_{n→∞}` 로 막대기(리만 합)를 무한히 더하면 곡선 아래 넓이 = 쌓인 총량.
   미분 "내리고 줄이고" ↔ 적분 "올리고 나누고".
4. **기본정리 = 둘은 서로 역.** "넓이가 자라는 속도 = 그 자리의 높이"(`A'(x)=g(x)`) 라서, 부정적분만
   찾으면 넓이가 `A(b)−A(a)` 로 공짜. 막대기 안 세도 됨.
5. **미적분 = 정적 산수를 `lim` 으로 감싸 동적 세계로 연장한 것.** 뿌리는 `lim` 하나.

## 직접 손유도한 것 (흔적)

- `x² → 2x`, `x³ → 3x²` — 정의에서 4박자(전개→상쇄→약분→h→0). — `THEORY.md` 파트 2
- `xⁿ → n·xⁿ⁻¹` 일반 증명 (h 를 딱 하나 문 항만 생존). — 파트 3
- 상수배·합 규칙, 다항식 미분(`3x²+5x+7 → 6x+5`). — 파트 3
- `∫₀³ 2x = 9` 를 **막대합 / 삼각형 / 부정적분** 세 방법으로 일치 확인. — 파트 4
- 기본정리 `A'(x)=g(x)` 유도. — 파트 5

## 막혔던/헷갈린 지점 (그리고 해소)

- **"지수 100 이 변수인가?"** → 아니다. 지수는 **고정(기계의 모양)**, 변하는 건 **입력 x**. (개념 박스로 박음)
- **"가로/세로만 따지면 2D 전용인가?"** → 아니다. 가로/세로는 입문 무대일 뿐. 미분의 심장(입력을 밀 때
  출력 반응)은 차원을 안 가림 → 입력 여러 개면 편미분·그래디언트(딥러닝 엔진).
- **"값 자체를 예측?"** → 아니다. 값은 원래 함수가 줌. 미분이 주는 건 **변화 속도**.

## 12살 설명 버전 (한 문단)

> 지금까지 배운 수학은 **멈춰있는 것**만 다뤘어(사각형 넓이, 일정한 속도). 근데 세상은 계속 변해.
> **미분**은 "지금 이 순간 얼마나 빨리 변하나"(곡선의 기울기)를, **적분**은 "조금씩 쌓인 걸 다 더하면
> 얼마나?"(곡선 아래 넓이)를 구하는 도구야. 둘은 곱셈·나눗셈처럼 **서로 반대**라, 하나를 알면 다른 하나를
> 얻어. 그래서 로켓을 쏘기 전에 어디 떨어질지 계산하고(예측), 원하는 데 떨어뜨리려면 어떻게 쏠지
> 역산할(제어) 수 있어. 이 모든 걸 가능하게 한 딱 하나의 아이디어는 **"0 에 닿지 말고 무한히 다가가라"(극한)**.

## 활용 프레임 — 예측 & 제어

- **예측:** 지금의 변화(힘·속도·증가율)를 **적분으로 쌓아** 미래를 미리 안다.
- **제어:** 원하는 미래를 정하고 **미분으로 역산해** 지금 뭘 할지 정한다.
- 로켓·자율주행·전염병·AI 학습 = 측정 →(미분) 어긋남 →(적분) 재예측 → 조정, 을 초당 수백 번 왕복.

## 한계 / 안 다룬 것

- 함수는 `x²·x³·2x` 최소셋만. 삼각함수·지수·로그 미분은 안 함(후속 후보).
- `lim` 을 ε-δ 로 엄밀히 정의하진 않음(직관 수준). 연속성·미분가능성 조건도 생략.
- 곱의 법칙·연쇄법칙·다변수(편미분/그래디언트)는 개념만 언급, 유도는 안 함.

## 후속 후보

- 삼각함수 `sin x` 미분을 극한으로 유도 (→ `cos x`)
- 곱의 법칙·연쇄법칙을 정의에서 유도
- 다변수로 확장: 편미분·그래디언트를 `y=x²+z²` 같은 곡면에서 (딥러닝 경사하강 연결)
- 미분방정식 맛보기: `y'=y` 가 왜 지수함수를 낳는지 (오일러 방법으로 한 스텝씩)
