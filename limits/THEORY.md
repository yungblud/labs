## 파트 1 — 함수의 극한

### 1-1. 극한의 뜻 — "닿지 말고 다가가라" (정식 버전)

미적분 노트에서 직관으로 본 걸 교과서 언어로 정리한다.

> `x` 가 `a` 에 한없이 가까워질 때 `f(x)` 가 일정한 값 `L` 에 한없이 가까워지면,
> **`x→a` 일 때 `f(x)` 의 극한값은 `L`** 이라 하고 이렇게 쓴다:
> ```
> lim_{x→a} f(x) = L
> ```

**가장 중요한 한 가지 — `x=a` 에서의 함숫값 `f(a)` 와 극한은 무관하다.**
극한은 "a 로 **다가가는 도중**" 값이 어디로 향하나만 본다. a 에 **닿았을 때**(x=a) 값이 무엇인지,
심지어 정의돼 있는지조차 상관없어. 예를 들어:
```
f(x) = (x²−1)/(x−1)
```
이건 `x=1` 에서 분모가 0 이라 **정의 자체가 안 돼**(함숫값 없음). 그런데도 `x→1` 극한은 멀쩡히 존재해:
```
(x²−1)/(x−1) = (x−1)(x+1)/(x−1) = x+1   (x≠1)
→  lim_{x→1} f(x) = 1+1 = 2
```
`x=1` 에 구멍이 뚫려 있어도, **다가가는 값**은 2. "함숫값 ≠ 극한값"의 대표 예다. (이건 파트 3 연속에서
다시 핵심이 된다.)

### 1-2. 좌극한 · 우극한 — 어느 쪽에서 다가가나

`a` 로 다가가는 길은 **두 방향**이야. 왼쪽(작은 쪽)에서 오는지, 오른쪽(큰 쪽)에서 오는지.

```
좌극한:  lim_{x→a⁻} f(x)     x 가 a 보다 작은 값으로 다가감
우극한:  lim_{x→a⁺} f(x)     x 가 a 보다 큰 값으로 다가감
```

> **극한값의 존재 조건: 좌극한 = 우극한.**
> 둘이 같아야(그 공통값이 L) `lim_{x→a} f(x) = L` 이 존재한다. 다르면 **극한 없음.**

**눈으로 확인 (코드 보조):** `f(x) = |x|/x` 를 `x→0` 에서 보자. 이건 `x>0` 이면 1, `x<0` 이면 −1 인 함수.
```
      x (왼쪽에서) |   f(x)          x (오른쪽에서) |   f(x)
--------------------------------------------------------
        -1e-01 |     -1              1e-01 |      1
        -1e-03 |     -1              1e-03 |      1
        -1e-06 |     -1              1e-06 |      1

좌극한 = −1,  우극한 = +1  →  서로 다름  →  lim_{x→0} 는 존재하지 않음
```
아무리 0 에 다가가도 왼쪽은 계속 −1, 오른쪽은 계속 +1. **좌 ≠ 우 → 극한 없음.** 그래프로는 0 에서
위아래로 **점프**하는 모양(파트 3 의 '점프 불연속').

### 1-3. x → ∞ 의 극한

`x` 를 한없이 **크게**(또는 작게) 보낼 때 `f(x)` 가 향하는 값. "아주 멀리 가면 함수가 어디에 정착하나".
```
lim_{x→∞} 1/x = 0        (x 가 커질수록 1/x 는 0 에 붙음)
lim_{x→∞} (2x+1)/x = 2   (멀리서 보면 +1 은 무시할 만함 → 2)
```
값이 하나로 안 정하고 무한히 커지면 **발산**: `lim_{x→∞} x² = ∞` (극한값이 "없다"의 한 형태).

## 파트 2 — 극한의 성질과 계산

### 2-1. 극한의 사칙연산

`lim f(x) = A`, `lim g(x) = B` 이고 둘 다 존재하면 (같은 `x→a` 아래):
```
lim (f ± g) = A ± B
lim (f · g) = A · B
lim (f / g) = A / B      (단, B ≠ 0)
lim (k · f) = k · A      (상수배)
```
즉 극한은 **각 조각의 극한으로 쪼개서 계산**해도 된다. 대부분의 극한은 그냥 `x=a` 를 대입하면 끝나.
```
lim_{x→2} (x² + 3x − 1) = 4 + 6 − 1 = 9   (다항함수는 대입이 곧 극한)
```
문제가 되는 건 **대입했더니 `0/0`, `∞/∞`, `∞−∞`** 처럼 값이 안 정해지는 경우 — 이걸 **부정형**이라 하고,
식을 **변형해서** 부정형을 푼다. 아래 3종이 교과서 단골.

### 2-2. 0/0 꼴 → 인수분해로 약분

분자·분모에 `x=a` 를 넣으면 **둘 다 0** 이 되는 꼴. 원인은 분자·분모가 **공통인수 `(x−a)`** 를 가져서.
그 공통인수를 약분하면 부정형이 풀린다.
```
lim_{x→1} (x²−1)/(x−1)
  = lim_{x→1} (x−1)(x+1)/(x−1)      ← 인수분해
  = lim_{x→1} (x+1)                  ← (x−1) 약분 (x≠1 이라 가능)
  = 2
```
> **미분과 판박이:** 파트 2 미분 유도에서 `(2xh+h²)/h → 2x+h` 로 약분했던 그 동작이 바로 이 `0/0`
> 풀이야. 미분의 `[f(x+h)−f(x)]/h` 자체가 `h→0` 일 때 `0/0` 부정형이거든.

### 2-3. ∞/∞ 꼴 → 최고차항으로 나누기

`x→∞` 에서 분자·분모가 **둘 다 무한대**로 가는 꼴. **분모의 최고차항**으로 위아래를 나누면, 낮은 차수
항들이 0 으로 사라져 값이 드러난다.
```
lim_{x→∞} (3x² + 2x) / (x² − 5)
  = lim_{x→∞} (3 + 2/x) / (1 − 5/x²)   ← 위아래를 x² 로 나눔
  = (3 + 0) / (1 − 0)                   ← 2/x, 5/x² → 0
  = 3
```
**요령(암산용):** 분자·분모 **최고차항의 계수 비**. 위 예는 최고차가 둘 다 x² → `3/1 = 3`.
- 분자 차수 > 분모 차수 → 발산(∞)
- 분자 차수 < 분모 차수 → 0
- 차수 같음 → 최고차 계수 비

### 2-4. ∞−∞ 꼴 · 무리식 → 유리화 (켤레 곱)

`√( ) − √( )` 나 `∞−∞` 처럼 빼서 안 정해지는 꼴은 **켤레를 곱하고 나눠**(유리화) 형태를 바꾼다.
`(A−B)(A+B) = A²−B²` 로 루트를 없애는 게 핵심.
```
lim_{x→∞} ( √(x²+x) − x )
  = lim_{x→∞} ( √(x²+x) − x ) · (√(x²+x) + x)/(√(x²+x) + x)   ← 켤레 곱
  = lim_{x→∞} ( (x²+x) − x² ) / ( √(x²+x) + x )                ← 분자 A²−B²
  = lim_{x→∞} x / ( √(x²+x) + x )
  = lim_{x→∞} 1 / ( √(1 + 1/x) + 1 )                           ← 위아래 x 로 나눔
  = 1 / (√1 + 1) = 1/2
```

### 2-5. 압착(샌드위치) 정리

직접 계산이 어려운 함수를, **위아래로 감싸는 두 함수**의 극한으로 가둬서 값을 정하는 방법.
```
g(x) ≤ f(x) ≤ h(x)  이고   lim g = lim h = L   이면   lim f = L
```
양쪽에서 `L` 로 조여오면 가운데 `f` 도 `L` 로 눌린다("샌드위치"). 진동하는 함수(예: `x·sin(1/x)` 를
`−|x| ≤ x·sin(1/x) ≤ |x|` 로 가둠) 같은 데서 쓴다. (교과서는 소개 수준, 삼각함수 극한에서 본격 활용.)
